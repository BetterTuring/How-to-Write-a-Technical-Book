# How-to-Write-a-Technical-Book
《如何写作一本 IT 技术书》涵盖主题如下：
- 我能写书吗，成为作者的先决条件
- 流程：出版一本书需要多长时间
- 目录：好的目录是畅销书的一半
- 文前：是书脸面的一部分
- 正文：让书变得富有生命力
- 销量：一本书能卖多少册
- 收入：一本书作者能赚多少钱
